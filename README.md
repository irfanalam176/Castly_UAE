# ⚡✨ CASTLY ✨⚡
### *Talent Marketplace (UAE)*

> A React Native app connecting **artists** and **clients** across the UAE — fast, modern, and beautifully real-time.

---

## 🌈 About Castly

**Castly** is a mobile platform built with **React Native** that enables hiring and booking of creative talent in the UAE.

Artists can showcase and apply for jobs such as:

- 🎭 Models  
- 🎤 Voice Artists  
- 🎬 Actors  
- 🎧 Freelancers / Creatives  

Clients can post jobs, discover talent, and manage hiring seamlessly.

---

## ⚡ Core Features

### 💼 Job System
- Post new job listings
- Browse available opportunities
- Detailed job view with requirements
- One-tap job application

### 🧑‍🎤 Artist Profiles
- Talent portfolio showcase
- Skills & category tagging
- Availability status
- Application tracking

### 💬 Real-time Chat
- Client ↔ Artist messaging
- Job-based conversations
- Instant communication flow

### 💳 Payments Integration
- Secure payment flow
- Milestone-based payouts
- Transaction history tracking

---

## 🌐 App Flow

1. 🔍 Browse Jobs / Talent  
2. 📄 View Job Details  
3. ✍️ Apply or Hire  
4. 💬 Chat in Real Time  
5. 💰 Process Payment
